# Raw Data

This folder is intended to store raw, unmodified datasets used for analysis.

Raw data files are not included in this repository due to licensing and redistribution considerations.

Source dataset:
- IBM Telecom Customer Churn Dataset
- Link: [https://www.kaggle.com/datasets/thedrzee/customer-churn-in-telecom-sample-dataset-by-ibm/data]

No transformations are applied to data in this folder.
